Getting started
Welcome to tspr
1
Install tspr in Claude Code
Register tspr as an MCP server in your IDE:
claude mcp add tspr -- node /path/to/dist/cli/index.js mcp
2
Configure your LLM provider
Open Settings → to set your provider and model aliases. By default tspr uses claude-subprocess (no API key needed if Claude Code is already authenticated).
3
Run your first test
Ask Claude Code (cc) to test your project:
Use tspr to test my project at /path/to/my-app
tspr will auto-generate tests, run them in a Docker sandbox, and surface failures with suggested fixes here.
